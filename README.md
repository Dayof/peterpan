# Project X

Projeto da Hackathon Globo 2017.

## Sistema

Executar
```
$ python run.py
```

Pacotes bases
```
$ bower install
```

Vigiar sass
```
$ grunt
```

## Ambiente

Instalar
```
$ bash install.sh
```

Criar
```
$ virtualenv env
```

Iniciar
```
$ . env/bin/activate
```

Sair
```
$ deactivate
```
